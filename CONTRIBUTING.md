# Contributing

Created by Chandan Pandey.

Thank you for contributing to Aegis!

## How to Contribute

1. Fork the repository.
2. Create a feature branch.
3. Use Conventional Commits (e.g., `feat: add xyz`, `fix: resolve abc`).
4. Submit a pull request.

## Code Style

- Follow PEP 8.
- Use f-strings.
- Prefer clear, modular functions.

## Reporting Issues

Use the GitHub issue templates for bugs and feature requests.
